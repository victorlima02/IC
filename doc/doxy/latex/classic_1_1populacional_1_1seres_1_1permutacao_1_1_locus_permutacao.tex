Calcula uma representação (expressão) numérica para essa característica. 

{\ttfamily Características} podem conter uma série de informações sobre si mesma, constituindo o globalmente o genótipo dos {\ttfamily seres} que as contem. Porém, uma expressão numérica única, sumarizando os efeitos mais relevantes dessa unidade de formação do {\ttfamily ser} em um único número, é necessária tanto para criar-\/se um modelo fenótipo, quanto para propósitos de processamento.
